# TI Piccolo Projects

A collection of embedded systems projects using TI C2000 (Piccolo) microcontrollers, focused on low-level hardware control and real-time applications.

## Features
GPIO, ADC, PWM, SPI, I2C interfacing
Interrupt-driven design
Register-level programming
Real-time control examples

* SD Card SPI interface
* RC-522 RFID SPI interface
* SCI Communications

## Requirements
TI Piccolo (C2000) board
Code Composer Studio (CCS)
Optional: C2000Ware

## Getting Started

git clone https://github.com/AsbestosLampshade/TI-Piccolo-Projects.git

Import project into CCS
Build and flash to the board
